# Update-AzureStorageUpload

## SYNOPSIS
This function renews the Azure storage upload to remain active

## SYNTAX

```
Update-AzureStorageUpload [-FileUri] <String> [<CommonParameters>]
```

## DESCRIPTION
This function renews the Azure storage upload to remain active

## EXAMPLES

### EXAMPLE 1
