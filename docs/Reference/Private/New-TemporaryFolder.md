# New-TemporaryFolder

## SYNOPSIS
This function creates a new temporary folder just like New-TemporaryFile

## SYNTAX
```powershell
New-TemporaryFolder [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
This function creates a new temporary folder just like New-TemporaryFile

## EXAMPLES

### EXAMPLE 1
